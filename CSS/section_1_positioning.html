Block elements allways begin on a new line and height, line-height, top-margin, bottom-margin can be specified. Example block elements:
  • div
  • p
  • h1
  • form
  • ul and li
Inline elements begins on the same line as siblings. Height, line-height, top-margin, bottom-margin cannot be set. Example inline elements:
  • span
  • a
  • label
  • input
  • img
  • strong
  • em
Inline-block elements are inline elements that can accept some block properties. It can fit on the same line as inline or other inline-block elements. Height, line-height, top-margin and bottom-margin can be set. Default width is 100% but also can be changed if needed. Example:
  • img
  • video
  • input
  • textarea
Positioning elements:
  • Relative positioning
  • absolute positioning
  • floating
By default every element has set position value static, but static elements doesn't accept any box offset - top, right, left, bottom. To set this properties position must be relative or absolute
