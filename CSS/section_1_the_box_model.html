Box model properties
  • width - sets width of content, sumarry width is sum of margins, border, paddings and content
  • height - same as width
  • margin - sets space outside border. Can be set individualy to top, left, right or bottom. Shorthand: top, right, bottom, left
  • padding - space inside border, but outside content, has the same properties as margin.
  • border - occupy space between padding and margin. Has properties
    • border-width
    • border-style i.e. dotted, dashed, double, solid
    • border-color
  • box-sizing - sets outer limit of the element, default is content-box, but can set dimensions to padding or border as well.
box-sizing: padding-box
display
  • inline - can be part of block element and other inline elements. Doesnt accept top and bottom padding or margin.
  • block - ocuppies whole width of page, doesnt allow other elements to be on left or right.
  • inline-block - creates block element that alco can be inline
  • none - browser wont render element like it doesn't exists
