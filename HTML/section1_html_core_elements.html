Elements:
  • paragraph: <p>
  • images: <ing>
    • src="link.com" or src="img.png"
    • alt="descripction of the img when is not loaded"
    • width and height
  • Headings:
    • h1: <h1> - should be only one on page - hoglights the most important topic
    • h2: <h2> - describes primarly blocks of content
    • h3: <h3> - describes less important blocks
    • h4: <h4> - default size is 16px same as <p>
    • h5: <h5>
    • h6: <h6>
  • hyperlinks: <a>
    • - Unvisited links are blue and underlined by default. Has attribute href that specifies destination of where it should open. When page is still in development process and proper link doesn't exists, href can take placeholder # f.e. href="#" By default links open in existing tab or window, but this can be set by setting target:
    • _self - default setting - opens in the same tab or window
    • _top - open link in full body of window
    • _parent - opens link in the parent frame
    • _black - opens in new tab / window
    • framename - opens link in named frame
