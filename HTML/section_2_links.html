• Anchor tags a href="#contact" /a div id="contact"
  • Relative paths - img src="images/logo.svg". To go folder above: src="../logo.svg"
  • Absolite paths - img src="http://logo.com/logo.png"
  • Target link - typicaly open another page in same window. Default value is target="_self". Other values of target:
    • _self - default setting - opens in the same tab or window
    • _top - open link in full body of window
    • _parent - opens link in the parent frame
    • _black - opens in new tab / window
    • framename - opens link in named frame
