HTML main elements:
  • head: <head>
    • lang - helpful for search engines and screen readers
    • :root - single HTML element that contains evvery other HTML elements
    • metadata - character sets, document title, links to external files, css styles, js scripts
      • meta charset="UTF-8"
      • meta name="description" content="Tutorials"
      • link rel="stylesheet" href="css/layouts.css" type="text/css"
      • script type="text/javascript" src="script.js"
  • body: <body>
