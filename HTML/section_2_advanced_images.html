Image formats are divided into 2 categories - Raster and vector images:
Raster images:
  • GIF - graphics interchange format. Best for graphisc with flat color (max 256 colors) Can display simple animations
  • JPG, JPEG - Joint photographic experts group. Best for photography and muports milions colors, but compresses image, so some details are blured.
  • PNG - Portable Network graphics. No data los during compression, supports transparency.
Vector images:
  • SVG - Scalable vector graphics. Unlimited colors, supports transparency. Good for logos, icons or image patterns than can be resized
Image attributes
  • Image ALT img src="logo.png" alt="description of logo". Alt attribure is important for img elements, to help screen readers.
  • Image Title img src="logo.png" title="page official logo". Used with images and links, when mouse is over these elements, title will appear.
  • Image Width and Height img src="logo.png" width="90" height="100". Specifies size of displayed image.
  • Image Map. Map element is used for creating links on selected portions of an image, used with area. Clickable element has 3 shapes:
    • rect - rectangle. rect: shape="x1,y1,x2,y2" - left corner, top corner, right corner, bottom corner.
    • circle - circle: shape="x,y,r" - x and y cords, and radius
    • poly - poly: shape="x1,y1,x2,y2,x3....,xn,yn" . first and last cordinate must be the same to close a plygon
