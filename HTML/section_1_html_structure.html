HTML structure
  • element: <a href="link.com"> Display name </a>
  • opening tag: <a href="link.com">
  • end tag: </a>
  • attrubute: href="link.com"
  • attribute name: href
  • attribute value: "link.com"
Elements:
  • paragraph: <p>
  • anchors: <a>
  • headers: <h3>
  • script: <script>
Tags in HTML
  • <html> : defines an HTML document
  • <body> : Document body
  • <p> : paragraph text
  • <h1> : heading text
Self closing elements
  • <img> : image insert
  • <br> : line break
  • <hr> : horizontal rule
