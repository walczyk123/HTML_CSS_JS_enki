Form element is used to represent a section of a document that contains interactive controls for a user to input information. I.e. checkboxes, text fields radio buttons. Form attributes:
  • action - used to specify where to send a data. Two methods used to send the data - GET and POST
  • autocomplete - used to help indicate input values
  • name - specifies the name to identify a form
  • novalidate - form shouldn't be validated
