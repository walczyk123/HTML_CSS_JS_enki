Meter attributes
  • form - is used to associate meter with form element. The form is owner of meter.
  • min - minimum value of the meter. Default is 0, must be lower than max.
  • low - specifies value considered as low. It must be higher than min and smaller than high and max.
  • optimum- optimum value that must be in range from min to max.
  • value - current attribute value. Must be in range from min to max. If is out ouf this boundaries, it takes the closest min or max value.
  • high - specifies value considered as high. It must be higher than min and low, but smaller than max. If value is greater than max, then it takes max value
  • max - max value of the meter, default value is 1. Must be higher than min.
