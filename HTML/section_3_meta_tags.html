Meta tags do:
  • Some specific data will display in search engine results
  • Some of the tags will help improwe the pages SEO
  • Are used to provide information to all sorts of clients
  • Google will ignore meta tags it doesn't recognize
  • Are added to the head of nearly every web page
Meta tags:
  • charset - declare character encoding of particular page
  • name=
    • "description" content="Short summary"
    • "keyword" content="HTML, XHTML, web terms" . Today is almost unused.
    • "author" content="xyz". No effect in SEO. Some CMS use it.
    • "robots" content=
      • "none" - equivalent to "noindex" and "nofollow"
      • "noimageindex" - crawlers not index any image on page
      • "noarchive" - do not cache content of this page
      • "nocache" - same as "noarchive"
      • "nosnippet"
      • "nodp"
  • http-enquiv=
    • "refresh" content="30" . Refresh page every 30 secs
    • "refresh" content="30, URL=page.html" . Redirect to page.html after some time better: script window.location = "http://page.com"
    • refresh attributes:
      • content-language specify default language. Better use "lang"
      • content-security-policy specify content policy of current page
      • content-type - now charset
      • set-cookie - now http header Set-cookie
