span is used to style inline elements
div is used as block element
hr is used to create thematic break between paragraphs
Button attributes:
  • value - initial value of button
  • disabled -display disabled button
  • form - used to associate button with form, form value must be the same as id of the form
  • name - gives button a name, submited with forms data
  • type - type of button:
    • submit - submit forms data to the server (default if not set)
    • reset - reset values in form
    • button - when you want to trigger some scripts
