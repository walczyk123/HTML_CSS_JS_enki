Ordered list - used to represent an ordered list of items. Displays as a block with numerals, letters or Roman Numbers. ol has 3 attributes:
  1. type
    • a - lowercase letters
    • A - uppercase letters
    • i - specifies lowercase Roman numerals
    • I - specifies uppercase Roman numerals
    • 1 - specifies numbers
  2. reversed - just reversed
  3. start - start number. start="3", start="III", start="c"
Unordered lists, has 4 styles:
  • style="list-style-type:disc"
  • style="list-style-type:circle"
  • style="list-style-type:square"
  • style="list-style-type:none"
ul can be nested in other lists
Nested lists

  2. second ordered
    • first unordered
    • second unordered
    • third unordered
  3. third ordered
